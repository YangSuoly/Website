2 Management

2.1 Switch user

  • su

即 switch user，切换用户。用户管理需要以管理员身份执行，所以，要先切换账户到 root

2.2 User operation

  • useradd
  • passwd
  • userdel

即 user delete

2.3 Group operation

  • groupadd
  • groupdel
  • useradd
  • usermod
  • cat

2.4 File permission

  • View Permission
  • chmod

即 Change file mode，修改文件的访问权限

  • chown

即 Change owner，修改文件的属主，一般每个用户只操作自己用户目录

2.5 Script
