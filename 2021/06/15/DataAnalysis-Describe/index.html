SibSp => 堂兄弟/妹个数
Parch => 父母与小孩个数

Fare => 票价
Cabin => 客舱
Embarked => 登船港口

【思考】所谓将表头改为中文其中一个思路是：将英文列名表头替换成中文。还有其他的方法吗？
